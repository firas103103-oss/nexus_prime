# MRF103 ARC Namer CLI

The MRF103 ARC Namer CLI is a command-line interface tool designed to facilitate the naming process within the MRF103 ecosystem. This tool provides a set of commands that can be used to generate, manage, and manipulate names for various components in the MRF103 framework.

## Features

- **Command Line Interface**: Easy to use CLI for quick access to naming functionalities.
- **Custom Commands**: Extendable command structure to add new naming commands as needed.
- **TypeScript Support**: Built with TypeScript for type safety and better development experience.

## Installation

To install the MRF103 ARC Namer CLI, you can use npm:

```bash
npm install -g mrf103-arc-namer-cli
```

## Usage

After installation, you can use the CLI by running:

```bash
arc-namer <command> [options]
```

### Commands

- `generate`: Generate a new name based on the provided criteria.
- `list`: List all generated names.
- `delete`: Delete a specific name from the records.
