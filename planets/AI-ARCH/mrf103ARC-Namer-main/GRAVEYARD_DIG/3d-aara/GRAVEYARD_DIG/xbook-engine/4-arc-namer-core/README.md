# MRF103 ARC Namer Core

The **ARC Namer Core** is a foundational library designed to facilitate the naming conventions and structures used within the MRF103 ecosystem. This library provides essential utilities, types, and configurations that are leveraged by other components of the MRF103 project.

## Features

- **Type Definitions**: Strongly typed definitions to ensure consistency across the ecosystem.
- **Utility Functions**: A set of helper functions to streamline common tasks related to naming.
- **Configuration Management**: Centralized configuration settings for easy adjustments and scalability.

## Installation

To install the ARC Namer Core, use npm or yarn:

```bash
npm install arc-namer-core
```

or

```bash
yarn add arc-namer-core
```

## Usage

Import the necessary modules from the ARC Namer Core in your project:

```typescript
import { someUtilityFunction } from 'arc-namer-core/utils';
import { SomeType } from 'arc-namer-core/types';
```
