# MRF103 ARC Namer - Visual Studio Code Extension

This project is a Visual Studio Code extension for the MRF103 ARC Namer. It provides functionalities to enhance the development experience when working with the ARC Namer ecosystem.

## Features

- **Command Palette Integration**: Easily access commands related to the ARC Namer.
- **Custom Providers**: Integrate with various providers to enhance functionality.
- **Type Definitions**: Strongly typed interfaces and types for better development experience.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/firas103103-oss/mrf103ARC-Namer.git
   ```
2. Navigate to the `arc-namer-vscode` directory:
   ```
   cd arc-namer-vscode
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage

- Open the command palette (Ctrl+Shift+P) and type in the commands related to the ARC Namer.
- Use the extension to streamline your workflow and improve productivity.

## Development

To contribute to the development of this extension:

1. Make sure you have Node.js and npm installed.
2. Follow the installation steps above.
3. Run the extension in the development mode:
   ```
   code --extensionDevelopmentPath=.
   ```

## Testing

Run the tests using Jest:
```
npm test
```
