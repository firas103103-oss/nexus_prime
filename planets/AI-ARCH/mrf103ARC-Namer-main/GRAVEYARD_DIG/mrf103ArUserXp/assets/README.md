# AR Assets Directory

This directory contains 3D models and textures for the AR experience.

## File Structure
- `diamond.gltf` - Diamond 3D model
- `crystal.gltf` - Crystal 3D model
- `textures/` - Texture files for materials

## Adding Models
To add new 3D models:
1. Export models in GLTF format
2. Place in this directory
3. Reference in the HTML assets section
4. Use in AR scenes as needed

## Supported Formats
- GLTF/GLB for 3D models
- PNG/JPG for textures
- OBJ (with MTL files)
