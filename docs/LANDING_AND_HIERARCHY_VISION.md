## 1. المستوى الأول: اللانينق الرئيسي (mrf103.com)

### الموقع
- **النطاق:** mrf103.com (القابضة والبراند الرئيسي)
- **الوظيفة:** نقطة الدخول الأولى — تعريف، رؤية، انطلاق

### متطلبات التصميم (أساسية)
| البند | الوصف |
|-------|-------|
| **الفخامة** | استثنائية التصميم — ليست عادية |
| **الحيوية** | رسومات، حركة، حياة — الموقع "يتنفس" |
| **الاحترافية** | واجهة متقدمة جداً |
| **الوضوح** | الرؤية السيادية بشكل واضح ومباشر |
| **اللغة** | مبسطة للجميع — لا تعقيد |
| **الزخم** | استعراض شامل يخلق انطباعاً قوياً |

### المحتوى المطلوب (ترتيب العرض)
1. **الرؤية السيادية** — ما نؤمن به وننطلق منه
2. **النواة** — كيف يعمل النظام (جينوم، هرمونات، Raqib/Atid، Soul Monitor) — راجع [NEXUS_PRIME_TECHNICAL_DOCUMENTATION_AR.md](NEXUS_PRIME_TECHNICAL_DOCUMENTATION_AR.md)
3. **التخصصات** — مجالاتنا
4. **المبادرات** — ما نطلق
5. **الإنجازات** — ما حققناه
6. **الزخم العام** — نبض الإمبراطورية

### الهدف
بعد استعراض كل شيء — الزائر يفهم، يقتنع، ويريد الانطلاق. من هنا ننطلق إلى الهيكلة التفرعية.

## 6. مقارنة: الوضع الحالي vs الرؤية

| البند | الوضع الحالي | الرؤية |
|-------|--------------|--------|
| اللانينق | موجود — تصميم جيد | ترقية لفخامة واستثنائية أعلى |
| الرؤية السيادية | مذكورة بشكل عام | واضحة ومباشرة بلغة مبسطة |
| التفرع | روابط منتشرة | هيكلة تفرعية شمولية واضحة |
| دار النشر | ✅ publisher | ✅ نفس الرابط |
| النظام التشغيلي | مبعثر (sovereign, platform, god) | توحيد تحت مفهوم واحد |
| اكس بيو | ✅ xbio | ✅ نفس الرابط |
| الرابع | boardroom, clone-hub منفصلين | تجميع تحت "الرابع" |
