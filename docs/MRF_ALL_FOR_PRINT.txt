────────────────────────────────────────────────────────────────────────────────
1.1 التعريف
────────────────────────────────────────────────────────────────────────────────

• الاسم: MRF / MrF / MRF103
• الدور: قابضة — الهوية التجارية الأساسية
• الموقع: mrf103.com
• النطاق: كل المنتجات والمشاريع والخدمات التابعة

MRF تمثل:
  - المالك: أنا كمالك للعلامة والمشاريع
  - المنتج: كمنتج وخدمة مستقلة أو مدمجة
  - المحور: المركز الأساسي لـ NEXUS PRIME والنظام البيئي
  - المستخدم: سواء للمالك أو لـ NEXUS أو لـ MRF أو للمستخدم النهائي (بيع، اشتراك، إلخ)
